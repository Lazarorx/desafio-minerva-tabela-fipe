## Versão 2: Mais Técnica (para audiência dev)

```
3 semanas. 2 versões. 1 objetivo: criar valor real.

Acabei de finalizar o Desafio Acelerado da Minerva com o FipeCheck - uma plataforma de análise inteligente de preços de veículos.

Stack técnica:
• Frontend: React 18 + Vite
• Visualização: Chart.js para análise comparativa
• APIs: Fipe (oficial) + scraping de 5 marketplaces
• Storage: LocalStorage (V1/V2) + PostgreSQL planejado (V3)
• Deploy: Vercel com CI/CD automático

Arquitetura evolutiva:
V1: MVP puro - consulta direta à API Fipe
V2: Análise de mercado + autenticação + dashboard
V3 (documentado): Worker Queue (Bull + Redis) para batch jobs

O diferencial? Não é só consultar a Fipe. É comparar o preço oficial com o mercado real em 5 fontes simultaneamente e mostrar se o veículo está acima ou abaixo da média.

Desafios técnicos resolvidos:
• Integração assíncrona com múltiplas APIs
• Normalização de dados de fontes heterogêneas
• Cálculo de desvio percentual em tempo real
• Componentização atômica (Atomic Design)
• Documentação técnica completa (diagramas, modelos de dados, casos de uso)

O projeto está 100% funcional e documentado, incluindo a visão de sistema completo com 5 atores operacionais (User, Researcher, Coordinator, Analyst, Admin) e suas responsabilidades.

Teste você mesmo:
```
